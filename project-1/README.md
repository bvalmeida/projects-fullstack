# Projeto Java 17 com Spring 3, Swagger, PostgreSQL 12 e Vue.js 3 - Aplicação Completa

Este é um projeto de exemplo que demonstra como criar uma aplicação completa usando as seguintes tecnologias:

- Java 17 com Spring 3 no backend
- Swagger para documentação da API
- PostgreSQL 12 como banco de dados
- Vue.js 3 no frontend
- Docker Compose para containerização de toda a aplicação

## Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas em sua máquina caso queira realizar alguma modificação:

- Java 17
- Maven
- PostgreSQL 12
- Node.js
- Vue CLI
- Docker e Docker Compose

## Execução

Caso deseje executar o projeto completo, execute o comando na raiz do projeto e ele será executado:

  ```bash
  docker-compose up -d
  ```

O backend estará disponível na porta 8000, o frontend na 8080.
Para acessar a doc da api backend, acesse local:port/swagger-ui.html
